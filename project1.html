Suspicious Activity Detection

Adaptive Multi-Pathway Deep Learning Framework

Project Overview

This project implements a real-time video-based human activity detection system to identify suspicious behavior. It combines YOLOv11 for object detection, pose estimation for tracking human movement, and I3D CNNs for spatio-temporal action recognition. The system integrates late fusion of multiple model outputs for enhanced accuracy.

The framework is optimized for speed and precision, enabling live monitoring of activities in video streams. It is implemented using Python, PyTorch, and OpenCV.

Key Features

The system includes:

  • Real-time video input processing
  • Human detection and tracking using YOLOv11
  • Pose estimation for accurate movement analysis
  • Spatio-temporal activity recognition with I3D CNN
  • Late fusion for high accuracy
  • Visualization of suspicious events
