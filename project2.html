Skin Cancer Classification

Deep Learning for Early Detection of Skin Cancer

Project Overview

This project focuses on building an AI model to classify different types of skin cancer from medical images. The system leverages convolutional neural networks (CNNs) for feature extraction and classification. It is designed to assist dermatologists in early detection and improve diagnostic accuracy.

The dataset is preprocessed using augmentation techniques to improve generalization, and the model is trained using Python, PyTorch, and medical imaging libraries.

Key Features

The project includes:

  • Preprocessing and augmentation of skin images
  • CNN-based classification of multiple skin cancer types
  • Evaluation using accuracy, precision, recall, and F1-score
  • Integration of Grad-CAM for visual explanation
  • Optimized training pipeline with PyTorch
